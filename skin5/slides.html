BitBucket : Point of View

BitBucket is a web-based hosting service for projects that use either the Mercurial or Git revision control systems. Bitbucket offers both commercial plans and free accounts. It offers free accounts with unlimited numbers of private repositories (which can have up to five users in the case of free accounts). Private repositories are not shown on profile pages. Bitbucket is written in Python using the Django web framework. It is similar to GitHub, which uses primarily Git.
